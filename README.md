В этом репозитории представлен аналитический проект, выполненный для агентства недвижимости. В рамках проекта было проведено исследование рынка жилой недвижимости в двух регионах с акцентом на временные характеристики объявлений и сезонную динамику.

Цель проекта: на основе данных об объявлениях выявить ключевые закономерности и дать рекомендации по повышению эффективности работы агентства недвижимости в Санкт-Петербурге и Ленинградской области.

Основные задачи:
1. Анализ времени продажи: определить, как площадь, стоимость, этаж и другие параметры влияют на срок актуальности объявлений в каждом регионе.
2. Анализ сезонности: выявить месяцы пиковой публикации новых объявлений и периоды наиболее активных сделок (снятия с продажи).
3. Сравнительный анализ: выявить ключевые различия между рынками СПб и ЛО по цене, типу жилья и динамике продаж.
4. Разработка рекомендаций: сформулировать конкретные предложения по планированию маркетинга, ценообразованию и распределению ресурсов с учетом выявленных закономерностей.
